# Exp - 2 LIBRARY MANAGEMENT SYSTEM

# AIM:
To create UML diagrams for a Library Management System.
# SRS (Procedure):
1. Open StarUML and create a new project named Library Management System.

2. Design the following UML diagrams: Use Case, Class, Activity, Package, Communication, and Sequence Diagrams.

3. Each diagram should represent system components, user interactions, and process flows clearly.

4. Save and export all diagrams, then include them in the report for documentation.
